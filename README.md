# ActionTP

Work in progress Third Person Action game in Unreal Engine 5

See the game in action: https://youtu.be/vUQnETxwbJ0 

# Key Features Implemented:
  - Custom character class
  - C++ Interfaces
  - Actor Components
  - UMG with Data Bindings
  - Animated UMG widgets
  - Attribute Component
  - Custom Materials
